Cone Survivor Game Design and Development

This is a two-week project I worked on for Prosper I.T Consulting.

On this project, I was given full creative control to design and develop any game I wanted using Unreal Engine 5, strictly using Blueprints and no C++. I chose to create a round-based survival first-person shooter inspired by classic arcade and modern wave shooters.

Using the cone shape as a pun for the title, I built much of the game’s aesthetic and character design around it — including cone-shaped enemies and weapons. I focused heavily on game features and systems, implementing modular controls to tweak gameplay variables during runtime, which allowed for smooth player stat upgrades and dynamic balancing.

To enhance progression, I added:

  • A vendor system for purchasing upgrades like fire rate, movement speed, and new weapons
  • A currency drop system from enemies
  • Multiple player boost mechanics tied to gameplay milestones

When I encountered a bug where enemies could pass through walls and floors, I took a creative approach instead of patching it out — I leaned into it and turned the enemies into ghosts, giving the game a unique twist and a nod to arcade nostalgia.

Sure, zombies are cool — but they’ve been overdone. I wanted something fresher... and a little more spectral. (Shout-out to One Slice Pizza Man, if you know, you know.)

I created modular spawner systems for both enemies and health pickups. These spawners can be placed anywhere in the level, and their behavior is fully customizable through the Details panel in the hierarchy — allowing you to set the item type and spawn intervals.
