Blender 3D Door Models

This was a personal project inspired by the work I was doing professionally at the time.

I noticed it was becoming difficult to distinguish between doorways, windows, and hallway openings—they all started to look like boxes with holes. So I took it upon myself to find a better way to create distinct, modular doors. I created a clever door modifier using Geometry Nodes and interchangeable parts.

These assets worked well with the scaled model I built. Here’s a procedural window modifier I created as well:

These shots demonstrate my attention to detail and the versatility of this package. By connecting and disconnecting nodes, I can switch hardware and parts to construct infinite door variations. Some modifications can even be made directly from the toolbar menu for easier editing.
